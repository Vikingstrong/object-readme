# Что такое Object?
Объект это переменая которая может хранить в себе свойства. С помощью объектов можно описать сущнсоть человека по его характеристикам

## Синтаксис объекта

``` js
    let user = {
        name: "userName",
        password: "querty123"
    };

```

"name" это ключ с помощью ключей мы можем обращаться к свойстам объектов а "userName" это свойство значение которое мы присвоили объекту

## Для чего нужны объекты?

Объекты прежде всего нужны для удобного хранения данных пользвотаеля. Данные пользователья так же можно хранить и в массиве однако контролировать такой массив значительно труднее чем объект

``` js
    let arr = ["userName", "querty123"];
              // 0          // 1
```
для того что бы брать данные из массива либо нужно брать их через индекс или присваивать их значения в переменую. По этому большее время используют объекты.

## Свойства и методы объектов

Если значения свойства это функция то это называется методом

``` js
    let obj = {
        sayhi: function(){
            console.log("Hi")
        };
    }
```

"sayhi" это свойство а его содержимое называется методом.
Вот как вызвать наш созданый метод

``` js
    obj.sayhi();
```
## Как получать значения из объектов

Значение из объектов можно получать с помощью 2 путей: Через . или Через []. К примеру:

``` js
    let user = {
        name: "Same",
        age: 22
    }
    console.log(user.name); // "Same"
    console.log(user["age"]); // 22
```

## Object.keys(); 🔑
Возарващает массив из ключей обьекта

``` js
    let obj = {
        key1: "hello",
        key2: "hi",
        key3: "bye"
    };
    console.log(Object.keys(obj)); // ["key1", "key2", "key3"]
```

## Object.values();
Возвращает значения объекта

``` js
    let obj = {
        user: "name",
        age: 25
    }
    console.log(Object.values(obj)) // ["name", "25"]
```

## Object.entries();
Возвращает массив и ключей и значений объекта

``` js
    let obj = {
        user: "name",
        age: 25
    }
    console.log(Object.entries(obj)) // [ ["user", "name"], ["age", 25] ]
```

## Перебор объекта for in

``` js
    let user = {
        name: "Jon",
        age: 24,
        password: "qwerty123"
    };
    for(let key in user){
        console.log(key);
    } // name, age, password
```
